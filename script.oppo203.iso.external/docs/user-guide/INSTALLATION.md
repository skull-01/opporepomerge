# Installation Guide

## Install from ZIP

1. Download the installable Kodi runtime ZIP for the release.
2. Open Kodi.
3. Go to **Add-ons**.
4. Choose **Install from zip file**.
5. Select the add-on ZIP.
6. Open the add-on settings after installation and complete the setup flow.

## Important notes

- Use the runtime ZIP for Kodi installation, not the dev-source ZIP or artifact bundle.
- The dev-source ZIP is for developers, testers, and maintainers.
- The artifact bundle is for release evidence and reconstruction.
- Hardware validation is not claimed unless a tester report exists for the exact hardware path.

## First safe test

Start with optional TV and AVR automation disabled. Confirm the base external-player handoff works first, then enable optional control paths one at a time.
