# Pre-Hardware Audit Report — v2.9.17 Final

The release is ready for user-owned hardware validation after software verification. The five new clone player variants and the cross-area Dolby Vision data layer all pass their software gates (full pytest 1219/3, 99% coverage, `mypy --strict` 51/0, ruff clean, release audit PASS, configurator gate green), but their real-device behavior is pending tester reports: the clone variants need exact-variant control confirmation on real M9205/M9702/optical-clone hardware, and the Dolby Vision stances need per-display validation with a UHD ISO / BDMV Dolby Vision FEL title (TV-led on full-DV TVs, Player-led/Auto on Sony/LLDV displays). Support-matrix wording separates software support from real hardware-validated support. Hardware validation is not performed and not claimed.
