# Release Notes — v2.9.10 Build 13

Build 13 adds software-tested Yamaha MusicCast/YXC AVR HTTP command support behind the optional AVR framework. The helper is guarded by existing AVR enablement and configuration validation, uses short timeouts, and returns non-fatal results for failure paths.

This build is software-verified only. Real Yamaha, Denon/Marantz, OPPO/Chinoppo, TV, AVR, Kodi, NAS, ADB, Roku, and SmartThings hardware validation was not performed or claimed.
