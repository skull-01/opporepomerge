# Release Notes — v2.9.1 Build 1

Version 2.9.1 Build 1 improves the first-run wizard wording for Kodi startup auto-power.

The wizard now explains that Kodi can automatically power on the OPPO/compatible player when Kodi starts, and that choosing No keeps the player off until playback starts or until the user powers it on manually.

No playback behavior, startup power implementation, OPPO command map, service interception, XML routing, NAS adapter, or hardware-control behavior changed.

Hardware validation was not performed and is not claimed.
