# Hardware Validation Status — v2.5.3 Build 6

## Status

Hardware validation is **not performed** and **not claimed** for Build 6.

## What Build 6 means

Build 6 is a software-verified pre-hardware release candidate. It is ready for user-owned physical validation, but it does not prove behavior on any real OPPO UDP-203/205, Chinoppo/M9702-family player, Kodi installation, NAS, TV, or ADB device.

## Tester-owned evidence required before claiming hardware pass

- Device model and firmware/build.
- Kodi platform/version.
- OPPO/Chinoppo IP and port.
- NAS path mapping source and expected player-visible path.
- Tagged 4K/UHD/2160p ISO/BDMV/MPLS positive test result.
- Tagged loose-video negative test result showing Kodi remains the player.
- Option 4 XML mode result if XML mode is used.
- TV/ADB switching result if enabled.
- Logs or observations for any failure.
