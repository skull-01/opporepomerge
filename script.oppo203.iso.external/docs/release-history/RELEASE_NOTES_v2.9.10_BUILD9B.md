# Release Notes — v2.9.10 Build 9B

Build 9B adds the guarded SmartThings experimental request helper and fake API tests. The helper requires explicit experimental acknowledgement and returns sanitized non-fatal results for configuration, authorization, HTTP, and network failures. Raw SmartThings tokens are not returned in diagnostics or helper results.

This remains software-verified only. Real SmartThings/Samsung TV hardware validation was not performed or claimed.
