# Release Notes — Version 2.9.0

Version 2.9.0 is a stable release rebuilt from the verified v2.5.3 Build 6 software candidate. It packages the completed 4K UHD disc-style interception line under a clean release identity.

## Included behavior

- Tagged 4K/UHD/2160p disc-style service interception.
- Conservative Option 4 playercorefactory XML routing for ISO, BDMV, and MPLS only.
- Loose/raw video files stay with Kodi even when tagged 4K/UHD/2160p.
- playercorefactory XML backup, idempotent merge, and rollback safety.
- Hardware-validation readiness export for tester evidence collection.

## Hardware validation

Hardware validation is not claimed in this release rebuild. Real device results must be provided separately before declaring hardware validation complete.
