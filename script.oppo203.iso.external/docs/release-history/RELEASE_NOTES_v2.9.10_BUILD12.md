# Release Notes — v2.9.10 Build 12

Build 12 adds software-only Denon/Marantz AVR command support for future optional AVR workflows. The driver remains behind AVR enablement and is not wired into playback sequencing.

Denon/Marantz support includes `PWON`, `SI<input>`, `PW?`, and `SI?` helpers with short timeout, open/close-per-command socket behavior, and non-fatal result handling.

Hardware validation is not claimed.
