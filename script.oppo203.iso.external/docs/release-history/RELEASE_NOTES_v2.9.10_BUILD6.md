# Release Notes — v2.9.10 Build 6

Build 6 adds the Android / Google TV preset pack for the v2.9.10 TV expansion phase. The added editable ADB software presets cover TCL, Sony, Hisense, Philips, Xiaomi, Sharp, Skyworth/Coocaa, Haier, and generic Android TV / Google TV.

No universal ADB HDMI command is claimed. Hardware validation is not claimed. Commands remain editable through the existing ADB command fields, and the established non-fatal TV switching behavior is preserved.

No playback routing, OPPO command-map payload, service interception, XML routing, NAS/AutoScript behavior, startup auto-power behavior, TV switching execution behavior, or AVR sequencing changed.
