# Hardware Validation — v2.9.17 Final

Hardware validation was not performed and is not claimed.

This is a software-verified release. Passing automated tests and release audits does not prove real OPPO, Chinoppo, clone, Magnetar, Reavon, Kodi, NAS, TV, or AVR hardware behavior. In particular the v2.9.17 additions — the five new clone player variants (M9205 V2/V3/V4, M9702 Plus, VenPro V203) and the cross-area Dolby Vision capability layer (per-player TV-led / Player-led stances and the global processing rule) — are research-sourced and software-verified only. The clone variants inherit their base devices' control behavior without exact-variant hardware testing, and the Dolby Vision stances are public-evidence recommendations, not measured device results. Every player row remains `validated: false`; real tester results (especially exact-variant Dolby Vision FEL validation via UHD ISO / BDMV) must be recorded separately before any path is described as validated.
