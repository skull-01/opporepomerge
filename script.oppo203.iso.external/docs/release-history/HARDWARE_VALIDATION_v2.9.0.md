# Hardware Validation — Version 2.9.0

Hardware validation is **not performed** and **not claimed** for the automated v2.9.0 release rebuild.

## Not tested here

- Real OPPO UDP-203/UDP-205 hardware.
- Real Chinoppo/M9702/M920x hardware.
- Real Kodi installation.
- Real NAS playback path.
- Real TV/ADB switching.

## Required tester evidence before claiming hardware pass

- Device model and firmware.
- Kodi version and platform.
- NAS/file path examples tested.
- Positive 4K UHD disc-style examples.
- Negative loose/raw video examples.
- playercorefactory XML mode result if used.
- Service interception mode result if used.
- Any OPPO/Chinoppo/TV/ADB logs or observed failures.
