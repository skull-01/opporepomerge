# Release Notes — v2.9.10 Build 5

Build 5 adds the TV backend registry and preset foundation for the v2.9.10 TV expansion phase. Existing TV switching APIs and backend names are preserved. The new preset registry is software-only and does not claim support for any TV model until real hardware validation is provided.

No playback routing, OPPO command-map payload, service interception, XML routing, NAS/AutoScript behavior, startup auto-power behavior, TV switching execution behavior, or AVR sequencing changed.
