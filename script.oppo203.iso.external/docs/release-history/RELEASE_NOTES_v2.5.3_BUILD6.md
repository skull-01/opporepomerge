# Release Notes — v2.5.3 Build 6

Build 6 is a pre-hardware release-candidate packaging freeze for the v2.5.3 line.

## Included software capabilities

- Tagged 4K/UHD/2160p disc-style service interception.
- Loose/raw video exclusion so MKV, MP4, M2TS, TS, VOB, and other raw video files stay with Kodi.
- Conservative Option 4 XML rules for `iso`, `bdmv`, and `mpls` only when the visible filename/path contains `4K`, `UHD`, or `2160p`.
- Safe `playercorefactory.xml` merge/backup/rollback behavior.
- Hardware-validation readiness report export for testers.
- Runtime package cleanup policy.

## Important caveat

Build 6 is software-verified only. Hardware validation is not claimed until real OPPO/Chinoppo/Kodi/NAS/TV/ADB results are provided by the tester.
