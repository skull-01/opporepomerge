# RELEASE NOTES — v2.9.1 Build 15

Build 15 adds transition-safe Babel/gettext extraction support for the development workflow. `tools/i18n_extract.py` is the new facade for localization template extraction, while `tools/make_pot.py` remains preserved as the deterministic fallback for one build.

This is a tooling and documentation build only. Runtime playback/control behavior is unchanged, and hardware validation is not claimed.
