# Hardware Ecosystem Support Matrix — v2.9.10 Build 5

## Player layer

Stock OPPO, Chinoppo-style clones, experimental clones, and OPPO-like successors retain the Build 4 support posture. Reavon and Magnetar remain warning-only unless hardware-proven.

## TV layer

| Backend | Build 5 status | Notes |
|---|---|---|
| adb | preserved existing backend | Editable command fields remain active |
| sony_bravia | preserved existing backend | PSK/HDMI-port path preserved |
| lg_command | preserved existing backend | Editable external command path preserved |
| samsung_command | preserved existing backend | Editable external command path preserved |
| custom_command | preserved existing backend | Generic escape hatch preserved |

Preset registry entries are software-only and require real hardware validation before confirmed support is claimed.

## AVR layer

AVR support is not implemented in Build 5. Future v2.9.10 AVR builds remain disabled by default.
