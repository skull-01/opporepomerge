# Code of Conduct

This project expects respectful, constructive collaboration.

## Expected behavior

- Be respectful and patient.
- Describe issues with enough detail to help reproduce them.
- Keep feedback focused on the code, documentation, tests, or hardware evidence.
- Respect that home-theater setups vary widely by firmware, wiring, NAS path style, TV model, AVR model, and Kodi platform.

## Unacceptable behavior

- Harassment, personal attacks, or discriminatory language.
- Publishing private credentials, tokens, PSKs, or personally sensitive network details.
- Misrepresenting untested hardware paths as validated.
- Pressuring maintainers or testers to make unsafe claims.

## Enforcement

Maintainers may edit, hide, or remove content that exposes secrets, misrepresents validation status, or violates respectful collaboration expectations.
